Ajax Day04：飞机大战 - 增加用户注册和用户登录
 * 用户注册
   * 功能分析：
     * 页面内容：
       * 一个表单：用于提交用户注册信息
         * 表单中元素：
	   用户名：用于登录及显示的
	   密码
	   确认密码
     * 表单验证：
       * 验证不能为空：用户名、密码及确认密码
       * 验证长度：
         * 用户名：6-18位
	 * 密码：6-12位
       * 验证内容：
         * 用户名：只能输入英文、数字
	 * 密码：只能数字
     * 验证用户名是否可用
     * 接收服务器端响应：
       * 注册成功：将页面跳转到登录页面（显示登录内容）
       * 注册失败：停留在注册内容，让用户重新注册
   * 功能实现：
     * 页面内容：
       * 一个表单：用于提交用户注册信息
       * 表单中元素：
	   用户名：用于登录及显示的
	   密码
	   确认密码
     * 页面格式化：
       * 利用table标签进行格式化
     * 页面的美化：jQuery UI
       * Dialog将表单进行美化
     * 表单验证：formValidator
       * initConfig()方法：初始化配置表单验证
         * validatorGroup：设置验证组，值为数字
	 * onSuccess：表单验证成功后的回调函数
	 * onError：表单验证失败后的回调函数
       * formValidator()方法：用于真正的表单验证逻辑
       * inputValidator()方法：用于验证input元素的值
       * compareValidator()方法：用于验证两个input元素值的比较结果
     * 验证用户名是否可用：提交到服务器端进行判断
       * 使用Ajax异步提交
       * 在什么时候提交用户名信息？
         * 在提交表单之前，进行用户名是否可用的验证
       * 使用formValidator插件的AjaxValidator()方法
     * 接收服务器端响应：
       * Ajax异步提交表单
